# generator-pypi

Yeoman generator for python package index.

It's a real pain to initialize a python project everyday!

## Usage

```sh
# install yo first
npm i -g yo

# install generator-pypi
npm i -g generator-pypi

# yo!
yo pypi

# Oh, yeah, all done!
```
